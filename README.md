- **Week 04 — (Coming soon)**

## Repository Structure
Each `week-xx` folder contains:
- `README.md` (project summary)
- `coversheet_weekx.pdf`
- `notebook_weekx.ipynb`
- `brief/` (project brief)
- `data/` (datasets, if allowed)
- `assets/` (images/screenshots used in reporting)
